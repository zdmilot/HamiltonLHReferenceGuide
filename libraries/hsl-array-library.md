# HSL Array Library

Include: HSLArrLib.hsl

&#x20;

| Data                                                                     | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| ------------------------------------------------------------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| <p> </p><p>2-dim. array of variables</p><p>2-dim. array of sequences</p> | <p> </p><p>The HSLArrLib declares two two-dimensional arrays in the global scope, one of type variable and one of type sequence. Both of the two-dimensional arrays have a fixed first dimension (row), which will be returned by the function VarArrUpperBound(). The second dimension (column) of the two-dimensional arrays may be set dynamically for each row. That means, each row represents a dynamic array.</p><p> </p><p>Example:</p><p>       column</p><p>            |</p><p>        [][][][][][][][]</p><p>row - [][][][][][][][][][][]</p><p>        []</p><p>        [][][][][][][][][][][]</p><p> </p> |

## &#x20;Runtime Errors

|             |                                                                                                                                                                                                                                   |
| ----------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Invalid Row | The library detected at the specified line an invalid row. The row must be an integer index that is greater than or equal to 0 and less than or equal to the value returned by VarArrUpperBound or SeqArrUpperBound respectively. |
