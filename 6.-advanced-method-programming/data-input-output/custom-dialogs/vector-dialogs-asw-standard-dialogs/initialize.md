# Initialize

## Syntax

Initializes the library and resources of the dialogs. (required)

## Arguments & Return Values

| Return Values          | range   | description               |
| ---------------------- | ------- | ------------------------- |
| ASWGLOBAL::BOOL::TRUE  | boolean | Initialization successful |
| ASWGLOBAL::BOOL::FALSE |         |                           |
