# HSL Liconic STX Return Codes

All of the functions have the following return codes:&#x20;

|                      |                                    |
| -------------------- | ---------------------------------- |
| HSLLiconicSTX::TRUE  | the function finished successfully |
| HSLLiconicSTX::FALSE | the function finished with error   |
